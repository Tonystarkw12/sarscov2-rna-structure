### 第3页：技术路线与方法

# 技术路线与方法

## 数据获取 🧬
- NCBI下载SARS-CoV-2参考基因组 (NC_045512.2)
- 提取RdRp、N、S、M、E关键基因序列
- 收集SARS、MERS等相关冠状病毒同源序列

## 结构预测 🔬
- **ViennaRNA RNAfold**：最小自由能预测
- **LinearFold**：长序列快速预测算法
- **多方法验证**：交叉对比提高准确性

## 功能分析 🧩
- **MAFFT多序列比对**：保守性分析
- **RNAplfold**：可及性预测
- **基序识别**：蛋白结合位点预测

### 预测流程图
```
基因组数据 → 基因提取 → 结构预测 → 保守性分析 → 位点预测 → 结果可视化
```

### 主要算法工具
- **ViennaRNA RNAfold**: 最小自由能算法
- **LinearFold**: 线性时间复杂度预测

- **forgi/matplotlib**: 专业可视化
